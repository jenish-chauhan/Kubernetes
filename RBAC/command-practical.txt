Core RBAC Resources (Very Important)

There are ONLY 4 RBAC resources you must remember:

Resource	        Scope	        Purpose
===================================================
Role	            Namespace	    Defines permissions in ONE namespace
ClusterRole	        Cluster-wide	Defines permissions across ALL namespaces
RoleBinding     	Namespace	    Binds Role → User/SA
ClusterRoleBinding	Cluster-wide	Binds ClusterRole → User/SA
